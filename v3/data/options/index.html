List of profiles

A comma-separated list of profiles. Each profile stores a unique state for all extensions. Use multiple profiles to manage extensions in bulk, enabling or disabling them as needed.

Navigate to chrome://extensions/shortcuts to assign a custom shortcut for each profile.

List of ignored Extensions

A comma-separated list of extension IDs to exclude. The enable or disable state of these extensions will remain unchanged.

To find an extension id, press the "Details to Clipboard" button on the popup interface of this extension.
